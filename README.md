# chouette-serveur
Script pour automatiser l'administration du serveur

### Installation
```shell
poetry install
```

### Configuration
Creer un fichier config.yml en vous inspirant de config.yml.example

### Execution
```shell
poetry run python mail.py
```

### mysql to postgresql
